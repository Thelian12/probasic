¿Que son las etiquetas semanticas?

Estas etiquetas sirven para definir la estructura de mi sitio web.

Tabla de etiquetas semanticas

ETIQUETA DESCRIPCION
<header> Define un contenedor para contenido introductorio o un grupo de elementos de navegación para un documento o sección.
<nav> Se usa en un sitio web para agrupar enlaces de navegación, como un menú principal, que permiten a los usuarios desplazarse entre secciones de la misma página o ir a otras páginas del sitio
<main> Se usa para marcar el contenido principal y único de un documento web
<footer> Representa el pie de página de un sitio web o de una sección específica dentro de él
<section> Define un apartado o sección temática y significativa dentro de un documento, agrupando contenido relacionado de forma lógica.
<aside> Se usa para incluir contenido que está tangencialmente relacionado con el contenido principal de una página web, como barras laterales, notas al margen, anuncios o enlaces relacionados
<article> Se utiliza para agrupar contenido independiente y autónomo, como una publicación de blog, una noticia o un comentario
