Métodos de clase

¿Qué es un método de una clase?

Simplificando mucho, un método es el nombre que recibe una función que existe dentro de una clase. Se utilizan para englobar comportamientos o funcionalidades relacionadas en conjunto con la clase y mediante las cuales podemos segmentar y separar en bloques de código.

Por ejemplo, en el siguiente fragmento de código definimos una constante text que contiene el "Manz". En la siguiente línea, ejecutamos el método .repeat(), que es una función que pertenece a todos los objetos que son definidos como , y que simplemente repite el texto el número de veces que le pasamos por parámetro:


 const  text = "Manz";
text.repeat();

¿Qué es un método?

En nuestro caso, hablamos de métodos cuando nos referimos a funciones que existen en el interior de una clase. Observa los siguientes ejemplos.

En este fragmento de código tenemos una función independiente:


function hablar () {
    return "Hola";
}

Esta función no está asociada a ningún otro elemento. Simplemente existe en el ámbito global de nuestro programa. Vamos ahora a convertirla en un método de la clase Animal (y por lo tanto, devolver en ella algo más acorde):


// Forma corta (recomendado)
class Animal {
    hablar() {
        return "Cuak";
    }
}

Observa que esto no es más una forma de incluir la función anterior dentro de nuestra clase Animal. Realmente, podríamos también hacerlo de la siguiente forma, sin embargo, la anterior es mucho más compacta y sencilla, por lo que es la que más se suele utilizar, pero quizás con esta te resulte más claro lo que se está haciendo dentro de la clase:


    // Forma larga
class Animal {
  hablar = function() {
    return "Cuak";
  }
}

Una vez declarado el método hablar() dentro de la clase Animal, podemos instanciar el objeto mediante un new Animal() y tener ese método disponible. Ten en cuenta que podemos crear varias variables de tipo Animal y serán totalmente independientes cada una:


    // Creación de una instancia u objeto (pato)
const pato = new Animal();
pato.hablar(); // 'Cuak'

    // Creación de otra instancia u objeto (donald)
const donald = new Animal();
donald.hablar(); // 'Cuak'

Observa que el método hablar() existe tanto en las variables pato como donald porque ambas son de tipo Animal. Al igual que ocurre con una función normal, se le pueden pasar varios parámetros al método y trabajar con ellos como venimos haciendo normalmente con las funciones.

Métodos comunes de String

Método Descripción Ejemplo
.length Longitud de la cadena "hola".length → 4
.toUpperCase() Convierte a mayúsculas "hola".toUpperCase() → "HOLA"
.toLowerCase() Convierte a mayúsculas "HOLA".toLowerCase() → "hola"
.trim() Elimina espacios al inicio y final " hola ".trim() → "hola"
.includes(sub) Verifica si contiene una subcadena "hola".includes("la") → true
.indexOf(sub) Índice de la primera ocurrencia "hola".indexOf("l") → 2
.slice(start, end) Extrae una porción "hola".slice(1,3) → "ol"
.replace(a, b) Reemplaza subcadena "hola".replace("o", "a") → "hala"
.split(sep) Divide en array usando separador "a,b,c".split(",") → ["a","b","c"]

Métodos comunes para números

Método/Función Descripción Ejemplo
Number.isInteger(num) Verifica si es entero Number.isInteger(4.5) → false
num.toFixed(d) Redondea y devuelve string con decimales (3.14159).toFixed(2) → "3.14"
parseInt(str) Convierte string a entero parseInt("123abc") → 123
parseFloat(str) Convierte string a decimal parseFloat("12.34xyz") → 12.34
Math.round(num) Redondea al entero más cercano Math.round(4.6) → 5
Math.floor(num) Redondea hacia abajo Math.floor(4.9) → 4
Math.ceil(num) Redondea hacia arriba Math.ceil(4.1) → 5
Math.random() Número aleatorio entre 0 y 1 Math.random()
Math.max(a,b,...) Devuelve el máximo Math.max(1,5,3) → 5
Math.min(a,b,...) Devuelve el mínimo Math.min(1,5,3) → 1

Métodos comunes de Array

Método Descripción Ejemplo
.length Longitud del array [1,2,3].length → 3
.push(element) Añade elemento al final [1,2].push(3) → [1,2,3]
.pop() Elimina y devuelve el último elemento [1,2,3].pop() → 3, array: [1,2]
.shift() Elimina y devuelve el primer elemento [1,2,3].shift() → 1, array: [2,3]
.unshift(element) Añade elemento al inicio [1,2].unshift(0) → [0,1,2]
.indexOf(element) Índice de la primera ocurrencia [1,2,3].indexOf(2) → 1
.includes(element) Verifica si contiene el elemento [1,2,3].includes(3) → true
.slice(start, end) Extrae porción del array [1,2,3,4].slice(1,3) → [2,3]
.splice(start, count, items...) Elimina/añade elementos [1,2,3].splice(1,1,9) → [1,9,3]
.forEach(callback) Itera sobre elementos [1,2,3].forEach(x => console.log(x))
.map(callback) Crea nuevo array transformado [1,2,3].map(x => x*2) → [2,4,6]
.filter(callback) Filtra elementos que cumplen condición [1,2,3,4].filter(x => x>2) → [3,4]
.reduce(callback, initial) Reduce array a un valor único [1,2,3].reduce((acc,x) => acc + x, 0) → 6
.sort() Ordena elementos (por defecto lexicográfico) [3,1,2].sort() → [1,2,3]
